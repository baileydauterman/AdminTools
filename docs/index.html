About

This project was created to be an easier way for System Administrators to perform actions on remote machines.


To load this module into memory on PowerShell startup run:

            
                if(-not (Test-Path $profile)) { New-Item -Type file -Path $profile -Force }
                Add-Content -Path $profile -Value 'Import-Module "path\AdminTools\0.1.1\AdminTools.psm1"'

Get-ComputerTrackingInfo

When you only have a machine name or IP address use this command to get more information about a machine. Returns the following: ComputerName, Uptime, MACAddress, SerialNumber, IPAddress, IPv4Gateway, Users


Usage

            
                #if machine is online 
                Get-ComputerTrackingInfo $ComputerName

                #returns
                ComputerName : Computer Name   
                Uptime       : Uptime of Remote Machine    
                MACAddress   : Mac of Remote machine
                SerialNumber : Serial Number of Remote Machine
                IPAddress    : IPv4 of remote machine
                IPv4Gateway  : Switch machine is plugged into 
                Users        : [PSCustomObject] of logged in users

Get-ComputerUptime

Retrieves the computer uptime for a target machine, grabs local uptime if machine is not specified.


Usage

            
                #if no machine is passed get local machine
                Get-ComputerUptime 

                #get remote machine uptime
                Get-ComputerUptime RemoteMachine

Get-DirectoryLastWriteTime

Pass a directory and specify is you want to recurse through the directory or not. Returns the last time a file was written to in this directory.


Usage

            
                Get-DirectoryLastWriteTime -Path \\path\to\dir -Recurse
                # or
                Get-DirectoryLastWriteTime -Path \\path\to\dir

Get-ErrorPrinterJobs

Takes the name of the printer server and returns the printers and print jobs that are in error states.


Usage

            
                Get-ErrorPrinterJobs -ComputerName printer-server

Get-InstalledSoftware

Takes a computer name or a list of computer names and returns a PSCustomObject for each computer and all the installed software.


Usage

            
                # retrieve software inventory 
                $outputs = Get-InstalledSoftware -ComputerName $computers

                # write software to a specified directory 
                foreach($o in $outputs) {
                    if($o.Connection -and ($o.SoftwareCount -ne 1)) {
                        New-Item "\\directory\$($o.Name).csv" -ItemType File
                        $o.Softwares | Export-Csv "directory\$($o.Name).csv" -NoTypeInformation
                    } else {
                        New-Item "\\directory\$($o.Name) - COULD NOT CONNECT TO.txt" -ItemType File
                    }
                }

Get-MacAddress

Usage

            
                Get-MacAddress -ComputerName computer01

                Get-MacAddress
                #returns 01:01:01:01:01

Get-RemoteUsers

Uses command line built in query user function and outputs to a PSCustomObject that can be iterated over.


Usage

            
                Get-RemoteUsers -ComputerName computer01

Get-UserFoldersToDelete

Uses Get-DirectoryLastWriteTime to enumerate over user folders determining which users haven't updated any files in the specified amount of days. Returns the user folders that can be deleted.


Usage

            
                Get-UserFoldersToDelete -ComputerName computer01 -Days 120

Restart-ComputersWeekly

Sends the restart command to a group of computers. Specificially works with Format-ComputersWeekly.


Usage

            
                #if the file is saved correctly in the above directory, you can get the ones for today 
                $computers = Format-ComputersWeekly -Today 
                $restartResults = Restart-ComputersWeekly $computers -Hours 5
                
                #if you want to specify a CSV 
                $computers = Format-ComputersWeekly -Path "\\path\to\csv\" 
                $restartResults = Restart-ComputersWeekly $computers -Hours 5
